Sobre Nós

Aqui na King Açaí, somos fanáticos por açaí e estamos aqui para compartilhar essa paixão com vocês! Nossa missão é servir o açaí mais delicioso, preparado sempre com ingredientes fresquinhos e de alta qualidade. Cada tigela que preparamos é feita com açaí 100% puro, combinado com os melhores toppings para criar uma explosão de sabores que você vai adorar. Seja para um lanche rápido ou uma sobremesa caprichada, temos algo especial para você. Nossas combinações são feitas para agradar a todos os gostos, desde os clássicos até os mais criativos. Por que Somos Diferentes?
  • Ingredientes fresquinhos e de alta qualidade.
  • Variedade: Tem opção para todo mundo.
  • Amor pelo que fazemos: Cada tigela é feita com carinho.
